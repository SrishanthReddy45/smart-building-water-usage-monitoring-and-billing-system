# Smart Building Water Usage Monitoring System

This project monitors building water usage and stores the data in a secure database.

## Features
- Water usage simulation
- Secure MySQL database
- Backend API using Flask
- Web dashboard display
- Auto refresh data

## Technologies Used
Python
Flask
MySQL
HTML
CSS
JavaScript

## Run Project

1 Install Python libraries

pip install flask flask-cors mysql-connector-python

2 Run simulator

python simulator.py

3 Run backend server

python app.py

4 Open frontend/index.html
